Prior to integrating intent data, please ensure that Unify is receiving website data by checking [Settings > Website Tag](https://app.unifygtm.com/dashboard/settings/integrations/website-tag) (see [support doc here](https://www.notion.so/Integrate-Unify-to-your-website-9d085875792e4ef5ba69e6ef81e7e553?pvs=21) if needed).

## How to integrate 6sense

Navigate to [Settings](https://app.unifygtm.com/dashboard/settings/my-profile) in Unify and complete the flow for 6sense.

1. 6sense - navigate to [Settings > 6sense](https://app.unifygtm.com/dashboard/settings/integrations/6sense) and complete the flow to connect your 6sense account to Unify
    1. Your API token can be found [here](https://abm.6sense.com/login?redirect=%2Fsettings%2Fintegration%2Fapitokenmanagement) in the 6sense dashboard
    2. Some customers do not have their 6sense api token enabled by default. If thatâ€™s the case you can reach out to support@6sense.com or your dedicated CSM with the following email:
