Prior to integrating intent data, please ensure that Unify is receiving website data by checking [Settings > Website Tag](https://app.unifygtm.com/dashboard/settings/integrations/website-tag) (see [support doc here](https://www.notion.so/Integrate-Unify-to-your-website-9d085875792e4ef5ba69e6ef81e7e553?pvs=21) if needed).

## Integrate Clearbit Reveal for intent data

Navigate to [Settings](https://app.unifygtm.com/dashboard/settings/my-profile) in Unify and complete the flow for Clearbit Reveal.

1. Navigate to [Settings > Clearbit](https://app.unifygtm.com/dashboard/settings/integrations/clearbit) and complete the flow to connect your Clearbit Reveal account to Unify
2. Your API keys can be found [here](https://dashboard.clearbit.com/api) in the Clearbit dashboard
