Prior to integrating intent data, please ensure that Unify is receiving website data by checking [Settings > Website Tag](https://app.unifygtm.com/dashboard/settings/integrations/website-tag) (see [support doc here](https://www.notion.so/Integrate-Unify-to-your-website-9d085875792e4ef5ba69e6ef81e7e553?pvs=21) if needed).

## Set up website intent data via Unify (powered by 6sense), Clearbit Reveal or 6sense

Navigate to [Settings](https://app.unifygtm.com/dashboard/settings/my-profile) in Unify and complete the flow for Unify (powered by 6sense), 6sense or Clearbit Reveal.

1. Unify intent (powered by 6sense) - navigate to [Settings > Unify Intent](https://app.unifygtm.com/dashboard/settings/integrations/unify-intent) and complete the flow to start receiving intent data from Unify
2. Clearbit Reveal - navigate to [Settings > Clearbit](https://app.unifygtm.com/dashboard/settings/integrations/clearbit) and complete the flow to connect your Clearbit Reveal account to Unify
    1. Your API keys can be found [here](https://dashboard.clearbit.com/api) in the Clearbit dashboard
3. 6sense - navigate to [Settings > 6sense](https://app.unifygtm.com/dashboard/settings/integrations/6sense) and complete the flow to connect your 6sense account to Unify
    1. Your API token can be found [here](https://abm.6sense.com/login?redirect=%2Fsettings%2Fintegration%2Fapitokenmanagement) in the 6sense dashboard
